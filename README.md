# student-management-system
Project Name: Student Result Management System

Mentor Name: Mr. Sunil Kumar Aralimara Channappa

Developer Name: Shubham Satish Kumbhar

Objective:
The main goal of this project is to simplify the way private class teachers manage their courses, students, and results. With this system, teachers can securely log in to manage student information, courses, and results. Students also have the ability to log in using their ID to view their own results anytime, making the process transparent and convenient for both teachers and students.

Software & Tools Used:

Programming Language: Python

GUI: Tkinter

Database: SQLite

Operating System: Windows 10

Additional Modules Installed During Development:

SQLite3 / DB Browser: For managing and maintaining the database.

Tkinter: Used for designing the entire GUI-based project in Code Runner.

Pillow: Installed to work with images and use advanced image features in the GUI.

Project Structure / Pages:

database1.db: The main database file for the entire project.

Dashboard & DashboardStudent: Separate dashboards for admin (teacher) and students.

Login & Register: Secure login and registration pages.

Course, Student, Result: Dedicated pages to manage each module individually.

InsideCoding: Contains all the validation logic and core backend code.

Key Highlights:

Secure login system for both teachers and students.

Easy management of courses and student records.

Students can instantly check their results online.

User-friendly and visually appealing interface.
